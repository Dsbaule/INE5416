A)

        Classes Primitivas:

            São classes que não são definidas a partir de outras classes, alguns exemplos são:

                Bounded = Classe que estabelece limites mínimos e máximos.

                Eq = Classe que trata os métodos de igualdade e desigualdade e instancia todos os tipos

                Enum = Classe que trata da enumerabilidade, define os métodos de operações sobre tipos ordenados

        Classes Secundárias:

            São classes definidas a partir de outras classes, alguns exemplos são:

                Ord = Classe que define métodos para os tipos de dados ordenados

                Num = Classe que define os métodos para operações com números

                Real = Classe que define os métodos numéricos de operações e instancia o tipo Int

B)

        As classes numéricas de haskell se baseiam na classe Num, subclasse de Eq, masnão de Ord, pois predicados não se aplicam a numeros complexos.
